## Features

- Optimized layout for faster design mockups
- Pre-loaded design templates
- Collaborative annotations
- Integrated with Figma for seamless workflows
- Supports multiple themes and customization

## How to Use

1. Open the application: [Baseline](https://baselineapp.figma.site)  
2. Log in using your Figma account.  
3. Choose a template or start a new project.  
4. Use the built-in tools to create and annotate designs collaboratively.  
5. Export projects directly back to Figma or share links.
